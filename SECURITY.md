# Security Policy

This is an experimental Godot/C# starter project. It is not designed for
production security boundaries, account systems, online multiplayer, or handling
private user data.

## Reporting Issues

If you find a security issue in the repository, open a GitHub security advisory
for `hex12345/Fully-ai-dangan-project` if available. If advisories are not
available, open an issue with enough detail to reproduce the problem and avoid
posting credentials or private data.

## Scope

In scope:

- Accidentally committed sensitive data
- Unsafe helper scripts or build instructions
- Dependency or supply-chain problems introduced by this repository
- Asset-loading behavior that can read unexpected local files

Out of scope:

- Vulnerabilities in unrelated third-party tools
- Production deployment hardening
- Issues caused by private modifications outside this repository
